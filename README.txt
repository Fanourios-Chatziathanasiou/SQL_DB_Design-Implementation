How to import the database and run the python app:

1. Create a database in Postgres named TechManiacs_DB.
2. Open the query tool and paste the contexts of Tables-triggers&functions-data_text_file.txt file and execute.
3. run the TechManiacs_DB.py python file and follow the console's steps.

1.Create a database in Postgres named Techmaniacs_DB.
